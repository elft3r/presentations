Container History

Container History

  • 1979 — chroot
    • Change the root directory of a running process
  • 2000 — FreeBSD jail
    • Similar to chroot, but with additional process sandboxing (filesystems, users, networks)
  • 2001 — Linux VServer
    • Similar jail mechanism for Linux

Container History

  • 2004 — Solaris Zones
    • Full user, process, and filesystem space
  • 2006 — Open VZ (Open Virtuzzo)
  • 2006 — Process Containers
    • Announced by engineers at Google to isolate and limit resource usage
  • 2007 — Renamed to Control Groups (cgroups)

Container History

  • 2008 — cgroups merged into Linux kernel 2.6.24
    • LXC (LinuX Containers) project created
  • 2011 — Warden (CloudFoundry)
  • 2013 — Google open-sourced LMCTFY (Let Me Contain That For You)
    • Core concepts contributed to libcontainer in 2015
