# Deploying Secure Landing Zones in Highly Regulated Sectors

## Presentation Overview

A presentation on deploying secure cloud landing zones in highly regulated sectors. Covers multi-account strategies, account vending, security controls, and lessons learned.

## Slide Order

Sections are loaded in this order (see `index.html`):

1. `header.html` — title slide
2. `jochen.html` — speaker intro (symlink to shared file)
3. `landing-zone.html` — landing zone concepts
4. `legal.html` — legal/regulatory requirements
5. `multi-account.html` — multi-account strategy
6. `account-vending.html` — account vending process
7. `security.html` — security controls
8. `learnings.html` — key learnings
